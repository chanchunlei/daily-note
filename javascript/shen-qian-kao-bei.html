    浅拷贝的原理和实现

    # 浅拷贝的原理和实现

    浅拷贝定义：

    创建一个新的对象，来接受你要重新复制或引用的对象值。如果对象属性是基本的数据类型，复制的就是基本类型的值给新对象；但如果属性是引用数据类型，复制的就是内存中的地址，如果其中一个对象改变了这个内存中的地址，肯定会影响另一个对象。

    # Object.assign：

    该方法可以用于js对象的合并等，其中一个用途是可以进行浅拷贝。方法的第一个参数是拷贝的目标对象，后面的参数是拷贝的来源对象（也可以是多个来源）。

    Object.assign的语法为Object.assign(target, ...source)

        let target = {};
    let source = {a: {b: 1}};
    Object.assign(target, source);
    console.log(target); //{a: {b: 10}}
    source.a.b = 10;
    console.log(source); //{a: {b: 10}}
    //浅拷贝，只复制的存储地址

    但是Object.assign有几点需要注意：

    1.它不会拷贝对象的继承属性； 2.它不会拷贝对象的不可枚举的属性； 3.可以拷贝Symbol类型的属性。

        let obj1 = { a:{ b:1 }, sym:Symbol(1)}; 
    Object.defineProperty(obj1, 'innumerable' ,{ //给obj1设置一个不可枚举属性
        value:'不可枚举属性',
        enumerable:false
    });
    let obj2 = {};
    Object.assign(obj2,obj1)
    obj1.a.b = 2;
    console.log('obj1',obj1); 
    console.log('obj2',obj2); //innumerable不会被拷贝

    # 下面实现一个浅拷贝

        const shallowClone = (target) => {
        //判断是否是基础数据类型
        if(typeof target === 'object' && typeof target !== null) {
            const cloneTarget = Array.isArray(target) ? [] : {};
            for(let prop in target) { //迭代key值，可枚举属性
                if(target.hasOwnProperty(prop)){  //不可出现不可枚举属性
                    cloneTarget[prop] = target[prop];
                }
            }
            return cloneTarget;
        }else {
            return target;
        }
    }

    # 深拷贝的原理和实现

    深拷贝定义：

    将一个对象从内存中完整的拷贝出来给目标对象，彻底实现内存上的分离，从堆内存中开辟一个全新空间存放新对象；新对象修改不会影响原对象。

    # JSON.stringify()配合JSON.parse()

    这个方法足以满足日常开发需求但是有些地方需要注意：

    拷贝的对象的值中如果有函数、undefined、symbol 这几种类型，经过 JSON.stringify 序列化之后的字符串中这个键值对会消失； 拷贝 Date 引用类型会变成字符串； 无法拷贝不可枚举的属性； 拷贝 RegExp 引用类型会变成空对象； 对象中含有 NaN、Infinity 以及 -Infinity，JSON 序列化的结果会变成 null； 无法拷贝对象的循环应用，即对象成环 (obj[key] = obj)。

    # 手写一个深拷贝

    	let deepClone = (target, hash=new WeakMap()) => {
		if(target.constructor === Date) return new Date(target);
		if(target.constructor === RegExp) return new RegExp(target);
		if(hash.has(target)) return hash.get(target);
		//获取到对象的所有属性描述符，包括原型链的属性;
		let allProps = Object.getOwnPropertyDescriptors(target);
		//Object.create创建一个新对象，使用原对象为新对象提供__proto__;
		//Object.getPrototypeOf方法返回指定对象原型对象（内部[[Prototype]]属性的值）;
		//创建新对象并指定原型对象，并添加属性描述符
		let cloneObj = Object.create(Object.getPrototypeOf(target), allProps);
		hash.set(target, cloneObj);
		for(let prop of Reflect.ownKeys(target)) {
			cloneObj[prop] = typeof target[prop] === "object" && target[prop] !== null ? deepClone(target[prop], hash) : target[prop];
		}
		return cloneObj;
	}
	
	//验证
	let obj = {
	  num: 0,
	  str: '',
	  boolean: true,
	  unf: undefined,
	  nul: null,
	  obj: { name: '我是一个对象', id: 1 },
	  arr: [0, 1, 2],
	  func: function () { console.log('我是一个函数') },
	  date: new Date(0),
	  reg: new RegExp('/我是一个正则/ig'),
	  [Symbol('1')]: 1,
	};
	Object.defineProperty(obj, 'innumerable', {
	  enumerable: false, value: '不可枚举属性' }
	);
	obj = Object.create(obj, Object.getOwnPropertyDescriptors(obj))
	obj.loop = obj    // 设置loop成循环引用的属性
	let cloneObj = deepClone(obj)
	cloneObj.arr.push(4)
	console.log('obj', obj)
	console.log('cloneObj', cloneObj)
