    # 思考题：

    1. 用什么样的思路可以new关键词？
    2. apply、call、bind、这三个方法之间有什么区别？
    3. 怎样实现apply、call、bind方法？

    # new原理

    new 关键词的主要作用就是执行一个构造函数，返回一个实例对象 根据构造函数的情况，来确定是否可以接受参数的传递

    那么new的过程中都做了什么事呢？

    创建了一个新对象； 将构造函数中作用域赋值给新对象（this指向新对象）； 执行构造函数中的代码（为这个新对象添加属性）； 返回新对象。

    注意：new指令之后，构造函数必然会返回一个新对象；如果定义了返回的对象那么new命令会直接返回这个新对象，而不是通过new执行步骤生成的this对象。如果return的不是一个对象，那么它还是会根据new关键词的执行逻辑，生成一个新的对象（绑定了最新this），最后返回出来。

    总结：new关键词执行之后总是会返回一个对象，要么是实例对象，要么事return语句制定的对象。

    # new的实现

    根据以上new的特性：

    让实例可以访问到私有属性； 让实例可以访问构造函数原型（constructor.prototype）所在原型链上的属性； 构造函数返回最后的结果是一个对象。

    	function _new(ctor, ...args) {
		if(typeof ctor !== 'function') {
			throw 'ctor must be a function'
		}
		let obj = new Object();
		obj.__proto___ = Object.creat(ctor.prototype);
		
		//返回值
		let res = ctor.apply(obj, [...args]);
		
		//可以返回非null的Object和function
		let isObject = typeof res === 'object' && res !== null;
		let isFunction = typeof res === 'function';
		
		//返回指定的对象，或者返回指定this的对象；
		return isObject || isFunction ? res : obj;
	}

    # apply & call & bind 原理介绍

    apply & call & bind 是挂在Function对象上的三个方法，调用者三个方法必须是一个函数。

    func.call(thisArg, param1, param2, param3, ....);
func.apply(thisArg, [param1, param2, param3, ...]);
func.bind(thisArg, param1, param2, ...);
    1
    2
    3

    其中 func 是要调用的函数，thisArg 一般为 this 所指向的对象，后面的 param1、2 为函数 func 的多个参数，如果 func 不需要参数，则后面的 param1、2 可以不写。

    这三个方法共有的、比较明显的作用就是，都可以改变函数 func 的 this 指向。call 和 apply 的区别在于，传参的写法不同：apply 的第 2 个参数为数组； call 则是从第 2 个至第 N 个都是给 func 的传参；而 bind 和这两个（call、apply）又不同，bind 虽然改变了 func 的 this 指向，但不是马上执行，而这两个（call、apply）是在改变了函数的 this 指向之后立马执行。

    let a = {
  name: 'jack',
  getName: function(msg) {
    return msg + this.name;
  } 
}
let b = {
  name: 'lily'
}
console.log(a.getName('hello~'));  // hello~jack
console.log(a.getName.call(b, 'hi~'));  // hi~lily
console.log(a.getName.apply(b, ['hi~']))  // hi~lily
//bind不会立即执行
let name = a.getName.bind(b, 'hello~'); 
console.log(name());  // hello~lily

    # apply 和 call 的实现

    Funciton.prototype.myCall = function(context, ...args) {
	context = context || window;
	context.fn = this;
	let res = context.fn(...args)
	delete context.fn;
	return res;
}

Function.prototype.myApply = function(context, args) {
	context = context || window;
	context.fn = this;
	let res = context.fn(...args);
	delete context.fn;
	return res;
}

    # bind的实现

    Function.prototype.myBind = function(context, ...args) {
	if(typeof this !== "function") {
		throw new TypeError('Error');
		let that = this;
		return function F() {
			if(this instanceof F) {
				//当被bind绑定过的方法被当成构造函数，使用new一个实例的时候，会忽略之前bind绑定的this，new将this指定为当前实例。
				//直接返回原始方法的实例即可；
				return new that(...args, ...arguments);
			}
			return that.apply(context, [...args].concat(...arguments))
		}
	}
}
